# Code Optimization Skill

## Overview

This skill specializes in analyzing and improving existing code for better performance, maintainability, and clarity.

## Capabilities

- **Performance Tuning**: Reduce time and space complexity
- **Refactoring**: Improve code structure and organization
- **Code Cleanup**: Remove duplication and improve readability
- **Modernization**: Update to use newer language features
- **Bottleneck Analysis**: Identify and fix performance issues

## Instructions

### 1. Analyze Current Implementation
- Review the code structure and logic
- Identify performance bottlenecks
- Note code smells and anti-patterns
- Check for security vulnerabilities

### 2. Optimization Strategies

#### Performance:
- Algorithm improvements (better time complexity)
- Data structure choices
- Caching and memoization
- Lazy loading
- Parallel processing

#### Readability:
- Better variable/function names
- Reduce complexity
- Improve organization
- Add documentation
- Remove magic numbers/constants

### 3. Provide Comparison

Show **before** and **after**:
```markdown
## Before
[original code]

## After
[optimized code]

## Improvements
- Performance: [X% faster]
- Readability: [improvements]
```

## When to Use

Use this skill when:
- User asks to "optimize", "improve", "refactor" code
- User mentions performance issues
- User wants to clean up or reorganize code
- User asks to make code more efficient
