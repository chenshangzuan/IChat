# Code Generation Skill

## Overview

This skill specializes in writing clean, efficient, and maintainable code across various programming languages and paradigms.

## Capabilities

- **Function Creation**: Write standalone functions with proper signatures
- **Class Design**: Create well-structured classes with appropriate methods
- **Module Organization**: Structure code into logical modules
- **Documentation**: Add clear comments and docstrings
- **Type Safety**: Include type annotations where applicable
- **Best Practices**: Follow language-specific conventions

## Instructions

### 1. Understand Requirements
- Identify the programming language
- Understand the functionality to implement
- Consider performance and maintainability
- Note any specific constraints or requirements

### 2. Write Code

#### Code Structure Checklist:
- [ ] Clear and descriptive names
- [ ] Proper indentation and formatting
- [ ] Error handling (try-catch, error propagation)
- [ ] Input validation
- [ ] Type annotations (if applicable)
- [ ] Documentation comments
- [ ] Following language conventions

#### Implementation Guidelines:
- **Keep it Simple**: Avoid over-engineering
- **DRY Principle**: Don't repeat yourself
- **SOLID Principles**: Single responsibility, open-closed, etc.
- **Performance**: Consider time and space complexity
- **Security**: Validate inputs, prevent injections

### 3. Provide Usage Example

Include example usage:
```python
# Example usage
result = function_name(input_data)
print(result)
```

## When to Use

Use this skill when:
- User asks to "write code", "create a function", "implement"
- User requests new functionality or features
